📷 SuperKupon — Profile Picture

3 versi profile picture. Pilih yang paling lo suka → screenshot → upload ke Instagram.

📸 Cara Screenshot Profile Picture:
  1. Tekan Ctrl + 0 dulu (pastiin zoom 100%)
  2. Tekan Win + Shift + S → pilih Rectangular Snip
  3. Drag pas di pinggir gambar profile (ikutin shadow violet)
  4. Otomatis ke-copy → buka Paint → Ctrl + V → Save sebagai PNG
  5. Transfer file PNG ke HP (via WhatsApp Web/Google Drive)
  6. Upload ke Instagram pas signup

💡 Tip: IG bakal crop jadi LINGKARAN. Versi gue udah desain center-friendly, gak bakal kepotong.
